### 테이블 생성하기

- 프로젝트 루트에 `sql` 디렉토리를 만들고 sql 쿼리 파일을 저장한다.

**회원 테이블 파일** (`sql/ddl.sql`)

```sql
drop table if exists member CASCADE;
create table member
(
    id bigint generated by default as identity,
    name varchar(255),
    primary key (id)
);
```
