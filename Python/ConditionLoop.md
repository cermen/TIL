# 조건문과 반복문

## 조건문

#### if 구문
```python
if 조건식:
    조건식을 만족하는 경우
```

#### if-else 구문
```python
if 조건식:
    조건식을 만족하는 경우
else:
    조건식을 만족하지 않는 경우
```

#### if-elif-else 구문
```python
if 조건식1:
    조건식1을 만족하는 경우
elif 조건식2:
    조건식1을 만족하지 않는 경우 중 조건식2를 만족하는 경우
else:
    조건식1와 조건식2를 모두 만족하지 않는 경우
```

#### if-in 구문

```python
if i in iter:
    iter에 i가 들어있을 경우
```

#### 삼항 연산자

```python
(참일 경우) if 조건식 else (거짓일 경우)
```
