# 정규표현식

. ^ $ * + ? {} [] \ | ()
. -> 무엇이든 한 글자를 의미
^ -> 시작문자 지정

* -> 0 or more
+ -> 1 or more
? -> 0 or 1
[abc] -> a 또는 b 또는 c 중 한 문자와 매치
[^0-5] -> not
^[0-5] -> 시작문자가 0~5 사이
문자클래스
\d -> 숫자의 자릿수
\D -> 숫자가 아닝 문자 매칭의 자릿수
\w -> 문자 + 숫자
\W -> 문자 + 숫자 아닌 문자와 매치
\s -> 공백
010-0000-0000
^\d{3}-\d{4}-\d{4}

- sub()
- match()
- search()
- findall()
- finditer()
